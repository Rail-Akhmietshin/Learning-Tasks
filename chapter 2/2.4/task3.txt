Можно ли использовать заглавные буквы для имени birthday? А для age? Или одновременно для обеих переменных?

const BIRTHDAY = '18.04.1982';
const AGE = someCode(BIRTHDAY);

- Я считаю, что можно использовать для обеих переменных. Хоть значение переменной age и вычисляется во время выполнения программы, его значение остается постоянным.
А для постоянных (const) значений обычно используется верхний регистр.
